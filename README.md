# Local setup

To run both frontend & backend apps locally

## Prerequisites

- node 16

### `npm install`

install all dependencies

### `npm start`

Chat should be accessible .\
 [http://localhost:3000](http://localhost:3000)

### `npm test`

run unit tests from all projects
